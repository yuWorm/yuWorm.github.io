Golang在Windows上的交叉编译

2021年3月11日 19:32
，归类于 笔记

golang的编译功能非常好用，可以在任何平台编译出所有任何平台的程序，下面为在windows上编译其他平台，cpu的可运行程序。

下面命令适用于cmd

1
2
3
4
5
6

set GOARCH=amd64 #设置生成的程序适用于的cpu型号，常用的有amd64，386，arm64，arm
set GOOS=linux #设置生成的程序适用于的系统
go build hello.go

##linux可以使用如下命令
GOARCH=amd64 GOOS=linux go build hello.go

如上即可
