Hello Hugo！

第一次使用Hugo，记录一下hugo的安装过程（环境为Windows）。

首先去Hugo Releases上下载hugo的程序，下载过来解压，把解压目录添加到环境变量，这样就可以直接在cmd中直接使用hugo命令了。

之后执行hugo new site BlogName创建站点，创建完成后，BlogName目录下就会出现hugo的程序结构，如下：

E:.
├─archetypes #文章类型目录
├─content #文章存放目录
├─data #数据目录
├─layouts #模板目录，优先级大于theme/layout
├─static #静态文件目录
└─themes #主题文件目录

hugo创建好程序目录后，是没有默认主题的，得自己安装主题才能使用。

所以先安装一个主题，我这里安装的是maupassant，安装方式如下：

git clone https://github.com/JokerQyou/maupassant-hugo.git theme/maupassant

git将代码克隆到theme目录，然后修改根目录的config.toml文件，在里面添加一行theme="maupassant"来指定主题。

配置好后，就可以使用hugo new post/HelloHugo.md创建文章了。

之后就可以在根目录执行hugo server -D即可启动本地服务，-D参数可以将草稿文章，及文章属性draft:true的文章。

访问 http://localhost:1313/即可本地访问你的博客了。

测试完毕，就可以在博客根目录使用hugo命令生成静态页面，静态页面输出在博客目录下的public目录中，然后进入public目录，使用git命令将其上传到github page或者gitte page都行。
